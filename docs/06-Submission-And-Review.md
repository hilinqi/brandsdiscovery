# 06 Submission and Review

## Forms
Submit Your Brand; Request a Brand/Product; Report Incorrect Information.

## Submit Brand Fields
Name, URL, category, country, description, main products, contact email; optional logo/social links.

## Request Fields
Type, category, description, country, optional budget/email.

## Workflow
New -> Reviewing -> Approved, Rejected or Needs Information.

## Duplicate Prevention
Normalize website domains and compare brand names and existing claims/submissions.

## Admin
English + Chinese helper labels, including Pending Submissions / 待审核提交.

## Acceptance
Validation, anti-spam, duplicate checks, review queues, brand creation from approved submission and status email work.
