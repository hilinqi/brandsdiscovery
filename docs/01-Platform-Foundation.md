# 01 Platform Foundation

## Goal
Create a fast, responsive, English WordPress website.

## Public Pages
Homepage, login, register, password reset, account shell, search shell and 404.

## Homepage
Header; hero; search; featured categories; featured brands; latest discoveries; buying guides; request CTA; submit-brand CTA; footer.

## Header
Logo, Brands, Categories, Guides, Search, Submit Your Brand and Login/Account.

## UI
Mobile-first, accessible, reusable components. Soft white background, deep navy primary, teal/emerald CTA and limited warm accent.

## Authentication
Use WordPress users. Social login may use a mature plugin, but custom modules depend only on WordPress identity, roles and capabilities.

## Acceptance
Theme activates; homepage works; mobile/desktop pass; authentication pages work; public interface is English only.
