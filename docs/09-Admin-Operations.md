# 09 Admin Operations

## Menus
Dashboard / 工作台; Brands / 品牌; Categories / 分类; Products / 产品; Claim Requests / 品牌认领申请; Submissions / 提交审核; Guides / 指南内容; SEO / SEO设置; Settings / 设置.

## Dashboard
Published brands, pending reviews, pending claims, pending submissions, broken links and recent Visit Store clicks.

## Operations
Brand/category/product CRUD; review/publish/pause/reject/merge; claim approval/rejection/request-info/revoke; submission review.

## Roles
Administrator, Editor and Moderator.

## Language Rule
Every custom admin menu, title, field, status, button, help panel and validation message uses English plus Chinese helper text.

## Acceptance
Full helper coverage, permissions, queues and basic activity logging work. Merchants cannot access WP Admin.
