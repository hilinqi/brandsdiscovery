# 05 Merchant Center

## Pages
Merchant login, registration, Claim Brand, claim status and dashboard. All English only.

## Claim Flow
Submit company/contact/evidence; admin verifies domain/business email/evidence; approve or reject; approved merchant gains ownership access.

## Dashboard
Dashboard, My Brand, Products, Submission Status, Basic Traffic and Account Settings.

Summary: publication status, completeness, profile views, Visit Store clicks, pending edits and recommended actions.

## Editing
Merchant changes enter review where required. Editorial fields remain platform-controlled.

## Permissions
Merchant manages owned brands only, cannot alter verification, cannot publish restricted changes and cannot access WordPress Admin.

## Acceptance
Registration/login/claim/review work; ownership permissions block unauthorized access; dashboard and edit workflow work.
