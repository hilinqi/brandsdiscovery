# 02 Brand System

## Fields
Name, slug, logo, cover, short/full description, official website, origin country, primary/secondary categories, markets, shipping regions, payment methods, return policy, support contact, social links, verification, publication and claim status.

## Status
Publication: Draft, Pending Review, Published, Paused, Rejected.  
Claim: Unclaimed, Claim Requested, Claimed, Suspended.

## Brand Page
Hero; logo/name/category; verification; summary; Visit Store; About; Why We Like This Brand; representative products; shipping/payment; social links; related brands/guides; claim CTA; report-information link.

## Visit Store
Tracked safe outbound redirect: record event, validate destination, redirect to official store. No transaction occurs on BrandsDiscovery.

## Admin Labels
English + Chinese helper text, including Brand Name / 品牌名称 and Official Website / 官方网站.

## Acceptance
Admin CRUD works; pages render; R2 images display; redirect is tracked and safe; related brands and claim CTA work.
