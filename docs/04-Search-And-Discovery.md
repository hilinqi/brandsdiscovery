# 04 Search and Discovery

## Scope
Brands, categories, representative products and guides.

## Interfaces
Homepage/header search, suggestions, results and useful empty state.

## Ranking
Exact brand match; text relevance; category relevance; published/complete profile; verification; basic engagement. Sponsored content must be disclosed and cannot override exact relevance deceptively.

## Empty Result
Related categories, popular brands and Request a Brand/Product form.

## Acceptance
All supported types are searchable; suggestions work; unpublished records are excluded; filters use structured attributes; empty results remain useful.
