# 07 SEO System

## Page Types
Brand, category and guide pages indexable. Search, account, admin and thin filter pages noindex.

## Fields
SEO title, meta description, canonical, index setting, Open Graph image and social description.

## Schema
Organization, Brand, Product showcase, Article, FAQ and Breadcrumb only when matching visible content.

## Internal Links
Brands link to categories, related brands and guides. Categories link to subcategories, brands and guides. Guides link to referenced brands/categories.

## Sitemap
Include published brands/categories/guides. Exclude private, search and thin pages.

## Integration
Rank Math may supply baseline SEO; SEO Toolkit supplies platform-specific relationships, metadata and schema.

## Acceptance
Metadata, canonical, sitemap, schema and internal links validate.
