# Dependency Map
Platform has no module dependency.  
Brand depends on Platform.  
Category/Attributes depend on Platform and Brand.  
Search depends on Brand and Category.  
Merchant depends on Brand and WordPress authentication.  
Submission depends on Brand, Category and Admin.  
SEO depends on Brand, Category and Content.  
Admin consumes Core and Merchant interfaces.  
Settings supports all modules.  
Deployment requires all modules complete.  
Developer/Release governs all modules.
